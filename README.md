# Power Outage and Mental Health

This repository contains Stata code for analyzing the impact of power outages on mental health hospitalizations.

## Project Description
This project examines how power outage exposure affects mental health outcomes using administrative hospitalization data and weather/grid infrastructure variables.

## Data
- HCUP SID (hospitalization data)
- NOAA weather data
- Power outage data (utility/grid sources)

## Code Structure
- replication_code_main_results.do: Main analysis and regression results
